Website design and typeface

This website, as simple as it is reflects my personal taste.
Clean, uncomplicated, zoomable and using proper typefaces.

No serifs for navigation and menu.
Preferably San Franciso from Apple, but Helvetica would do too.

And a good serif font for reading.
Like Apple's New York for the main body of text.

If you have no idea what I am on about, please ignore this page.
To use Apple's beautiful New York and San Francisco fonts on the web, you need to do the following:

<style>
 .ny {
  font-family: -apple-system-ui-serif, ui-serif, Georgia, serif;
 }

 .sf {
  font-family: -apple-system, BlinkMacSystemFont, Helvetica, Arial, sans-serif;
 }
</style>
